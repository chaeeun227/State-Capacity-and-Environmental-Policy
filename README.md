# Environmental Policy Analysis

This repository contains empirical policy analysis projects related to
environmental public goods, state capacity, and environmental protection.

## Research Topics
- State capacity and environmental protection
- Environmental public goods provision
- Cross-country environmental policy analysis

## Methods
- Data analysis using R
- Visualization with ggplot2
- Linear regression and non-linear specifications

## Data
- World Bank
- Cross-country governance datasets

## Status
This repository is under active development.
